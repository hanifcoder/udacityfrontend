HTML = Skeleton = Foundation. CSS = Skin = Style Javascript = Brain = Functionality & behavior. There are over 100 types of HTML tags that each serves a specific purpose.
There are block and inline elements.
Div tag is a can be used as text representation built right in this div. Also create a wraper element with span and line breaks with br.
There are self-closing elements for image, input and special way to create comments which can only be seen by developers in the code lines, these comments won't show on the client side rrendering.
DOM: Document Object Model is the model that our web document is being presented as.
DOM methods allow programmatic access to the tree and then you can change the document syle, structure or content. DOM is called the tree because it starts with a single parent item that branches out into children item hierarchically.
CSS is the execution of design, with it one can take control of hundreds of attributes to alter layout of the web pages.
There are three ways to work with selectors such as ids, class and type.
CSS Grid and Flex Box are critical to understand and pratice to design responsive layouts.
